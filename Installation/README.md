# K?stencode Werkbank - Installation

Dieses Verzeichnis enthält eine fertige Docker-Compose-Installation für Host und Faktura.

## Voraussetzungen
- Docker
- Docker Compose (v2)

## Installation
1. In dieses Verzeichnis wechseln.
2. Einmalig starten:
   ./setup.sh
3. Öffne im Browser:
   http://localhost:8080

## Stoppen
./shutdown.sh

## Versionen
- Host: 1.1.0 (kuestencode/host:1.1.0)
- Faktura: 2.1.0 (kuestencode/faktura:2.1.0)
- Rapport: 1.0.0 (kuestencode/rapport:1.0.0)
